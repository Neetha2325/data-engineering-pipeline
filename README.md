## Overview
This project demonstrates an end-to-end data pipeline using Python, Snowflake, and Airflow.

## Workflow
1. Extract data from CSV using Python
2. Perform data cleaning and transformation
3. Load data into Snowflake
4. Validate and aggregate using SQL
5. Orchestrate workflow using Airflow DAG

## Tech Stack
- Python (pandas)
- Snowflake
- SQL
- Apache Airflow
